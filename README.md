# PoncePretelin — documentación (Mintlify)

Sitio publicado: [poncepretelin.mintlify.app](https://poncepretelin.mintlify.app/)

Repo de **solo documentación**. El código del gestor vive en [Poncepretelin](https://github.com/UlisesPoncePretelin/Poncepretelin) (privado).

## Editar

1. Modifica `docs/product/*.mdx` o `docs.json`.
2. Push a `main` → Mintlify despliega automáticamente.

## Preview local

```bash
npx mintlify@latest dev
```
