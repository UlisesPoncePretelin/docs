## Stack

| Capa               | Tecnología                                        |
| ------------------ | ------------------------------------------------- |
| Backend            | FastAPI, SQLAlchemy, Alembic, PostgreSQL          |
| Frontend           | Next.js 15 App Router, TypeScript, TanStack Query |
| Orquestación local | Docker Compose (`:3002` / `:8002`)                |

## Tenancy

Datos clínicos filtrados siempre por `organization_id`. Tests de backend usan SQLite aislado en memoria (no Postgres de desarrollo).
