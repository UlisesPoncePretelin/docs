## URL

**Login demo:** `demo@poncepretelin.app` / `PoncePretelinDemo2026!`

## Ruta sugerida

1. **Evaluaciones → Pruebas diagnósticas** — sensibilidad, especificidad y fuente bibliográfica.
2. **Paciente → Episodio** — intake, evolución SOAP y motor de recomendaciones.
3. **Agenda** — citas y reserva pública por enlace.
4. **Mensajes** — bandeja y campañas (módulo comms).
